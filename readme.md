# n8n Self-Hosted

N8N instance for self-hosting on Render.

## Environment Variables Required

- `N8N_BASIC_AUTH_ACTIVE`: Set to true for basic authentication
- `N8N_BASIC_AUTH_USER`: Username for basic authentication
- `N8N_BASIC_AUTH_PASSWORD`: Password for basic authentication
- `N8N_HOST`: n8n-self-hosted-oydc.onrender.com
- `N8N_PROTOCOL`: https
- `N8N_PORT`: 3000
- `NODE_ENV`: production
